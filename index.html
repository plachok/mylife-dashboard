☁️ Sync & Backup
📱 ติดตั้งเป็น App บนมือถือ
iPhone / iPad (Safari)
1. เปิด link ใน Safari (ไม่ใช่ Chrome)
2. กดปุ่ม Share (กล่องมีลูกศรขึ้น) ด้านล่าง
3. เลือก "Add to Home Screen"
4. กด Add — แอปจะอยู่บน Home Screen
Android (Chrome)
1. เปิด link ใน Chrome
2. กด ⋮ มุมขวาบน
3. เลือก "Add to Home Screen"
🌐 Upload สู่ GitHub Pages (เข้าได้จากทุกที่)
1. สมัคร github.com (ฟรี)
2. กด New Repository → ตั้งชื่อ เช่น mylife-dashboard
3. Upload ไฟล์ index.html
4. ไปที่ Settings → Pages → Deploy from main branch
5. ได้ link: yourname.github.io/mylife-dashboard
✅ เปิดได้ทุก browser ทุกเครื่อง ฟรีตลอด
📤 Export ข้อมูล
บันทึกข้อมูลทั้งหมดเป็นไฟล์ เพื่อย้ายไปมือถือหรือ backup
📥 Import ข้อมูล
นำไฟล์ JSON ที่ Export ไว้กลับเข้ามา — ใช้ย้ายข้อมูลระหว่าง computer ↔ มือถือ
⚠️ Import จะ แทนที่ข้อมูลเดิมทั้งหมด — Export backup ก่อนเสมอ
📊 ข้อมูลที่บันทึกไว้
